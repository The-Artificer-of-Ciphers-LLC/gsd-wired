  <files>.goreleaser.yaml, Dockerfile.container, .gitignore</files>

    Create `.goreleaser.yaml` at repo root with these exact sections per D-15 through D-20 and research STACK.md patterns:

    ```yaml
    version: 2

    before:
      hooks:
        - go mod tidy
        - go generate ./...

    builds:
      - id: gsdw
        binary: gsdw
        main: ./cmd/gsdw
        goos:
          - darwin
          - linux
        goarch:
          - amd64
          - arm64
        env:
          - CGO_ENABLED=0
        ldflags:
          - -s -w
          - -X github.com/The-Artificer-of-Ciphers-LLC/gsd-wired/internal/version.version={{.Version}}
          - -X github.com/The-Artificer-of-Ciphers-LLC/gsd-wired/internal/version.commit={{.Commit}}
          - -X github.com/The-Artificer-of-Ciphers-LLC/gsd-wired/internal/version.date={{.Date}}

    archives:
      - id: gsdw
        builds: [gsdw]
        format: tar.gz
        name_template: "{{ .ProjectName }}_{{ .Os }}_{{ .Arch }}"

    checksum:
      name_template: "checksums.txt"

    changelog:
      sort: asc
      filters:
        exclude:
          - "^docs:"
          - "^test:"
          - "^chore:"

    signs:
      - artifacts: checksum
        cmd: gpg
        args: ["--batch", "-u", "{{ .Env.GPG_FINGERPRINT }}", "--output", "${signature}", "--detach-sign", "${artifact}"]
        stdin: "{{ .Env.GPG_PASSWORD }}"

    notarize:
      - macos:
          - sign:
              certificate: "{{.Env.MACOS_SIGN_P12}}"
              password: "{{.Env.MACOS_SIGN_PASSWORD}}"
            notarize:
              issuer_id: "{{.Env.MACOS_NOTARY_ISSUER_ID}}"
              key_id: "{{.Env.MACOS_NOTARY_KEY_ID}}"
              key: "{{.Env.MACOS_NOTARY_KEY}}"
              wait: true
              timeout: 10m

    homebrew_casks:
      - name: gsdw-cc
        repository:
          owner: The-Artificer-of-Ciphers-LLC
          name: homebrew-gsdw
          branch: main
          token: "{{ .Env.HOMEBREW_TAP_GITHUB_TOKEN }}"
        commit_author:
          name: goreleaserbot
          email: bot@goreleaser.com
        homepage: "https://github.com/The-Artificer-of-Ciphers-LLC/gsd-wired"
        description: "GSD + Beads/Dolt workflow orchestration for Claude Code"
        license: "MIT"
        skip_upload: auto
        postinstall: |
          system_command "/usr/bin/xattr",
            args: ["-dr", "com.apple.quarantine", "#{staged_path}"]

    dockers_v2:
      - id: gsdw
        ids: [gsdw]
        dockerfile: Dockerfile.container
        image_templates:
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}"
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:latest"
        build_flag_templates:
          - "--pull"
          - "--label=org.opencontainers.image.created={{.Date}}"
          - "--label=org.opencontainers.image.title=gsdw"
          - "--label=org.opencontainers.image.revision={{.FullCommit}}"
          - "--label=org.opencontainers.image.version={{.Version}}"
          - "--platform=linux/amd64"
        extra_files:
          - Dockerfile.container
      - id: gsdw-arm64
        ids: [gsdw]
        dockerfile: Dockerfile.container
        image_templates:
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}-arm64"
        build_flag_templates:
          - "--pull"
          - "--platform=linux/arm64"
        extra_files:
          - Dockerfile.container

    docker_manifests:
      - name_template: "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}"
        image_templates:
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}"
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}-arm64"
      - name_template: "ghcr.io/the-artificer-of-ciphers-llc/gsdw:latest"
        image_templates:
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}"
          - "ghcr.io/the-artificer-of-ciphers-llc/gsdw:{{ .Version }}-arm64"
    ```

    IMPORTANT NOTES:
    - The `notarize:` section (BLOCKER fix for DIST-03) enables Apple notarization per D-05, D-07, D-09. GoReleaser v2 requires this block with sign certificate and notarize API key credentials. The `wait: true` and `timeout: 10m` ensure the release waits for Apple's notarization service to complete (per D-09: 30-90 seconds typical). Env vars MACOS_SIGN_P12, MACOS_SIGN_PASSWORD, MACOS_NOTARY_ISSUER_ID, MACOS_NOTARY_KEY_ID, MACOS_NOTARY_KEY must be set for local macOS releases per D-06, D-08.
    - The `postinstall:` in homebrew_casks (per D-20) runs `xattr -dr com.apple.quarantine` as a Gatekeeper fallback if notarization is delayed.
    - dockers_v2 is alpha per D-15. If `goreleaser check` fails on dockers_v2, fall back to standard `dockers` + `docker_manifests` split approach. Test with `goreleaser check` and adjust.

    Create `Dockerfile.container` at repo root per D-18:
    ```dockerfile

    COPY gsdw /usr/local/bin/gsdw
    ENTRYPOINT ["/usr/local/bin/gsdw"]
    ```

    Update `.gitignore` to add GoReleaser artifacts:
