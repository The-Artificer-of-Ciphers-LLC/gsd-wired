    - Test: CheckAll returns OK status for bd when bd is on PATH
    - Test: CheckAll returns FAIL status for bd when bd is NOT on PATH and NOT in GOPATH/bin
    - Test: lookInGoPath finds a binary in $(go env GOPATH)/bin when not on PATH (per D-04, SETUP-04)
    - Test: CheckAll returns install instructions for each missing dep (per D-05)
    - Test: CheckAll checks all four deps: bd, dolt, Go, container runtime (per D-03)
    - Test: version extraction parses "dolt version X.Y.Z" and "bd version X.Y.Z" output
    - Test: container runtime detection tries docker then podman (Apple Container deferred to Phase 13)
  </behavior>
  <action>
    Create `internal/deps/check.go` with:

    1. Types:
       - `Status` type (string): "ok", "warn", "fail"
       - `Dep` struct: Name, Binary, Status, Version (string), Path (string), InstallHelp (string)
       - `CheckResult` struct: Deps []Dep, AllOK bool

    2. `CheckAll() CheckResult` function:
       - Checks bd, dolt, go, docker/podman in that order
       - For each: exec.LookPath first, then fallback to lookInGoPath (for bd, dolt)
       - If found, runs `{binary} version` to extract version string
       - If not found, populates InstallHelp with actionable commands:
         - bd: `go install github.com/steveyegge/beads/cmd/bd@latest` or `brew install steveyegge/tap/beads`
         - dolt: `brew install dolthub/tap/dolt` or `curl -L https://github.com/dolthub/dolt/releases/latest/download/install.sh | bash`
         - Go: `brew install go` or download from https://go.dev/dl/
         - Container runtime: `brew install --cask docker` or `brew install podman`

    3. `lookInGoPath(binary string) (string, error)` unexported function:
       - Runs `go env GOPATH` to get GOPATH
       - Checks filepath.Join(gopath, "bin", binary) exists via os.Stat
       - Returns full path if found, error if not
       - Per SETUP-04: this avoids false negatives after `go install`

    4. `checkBinary(name, binary string, installHelp string, tryGoPath bool) Dep` unexported function:
       - exec.LookPath first
       - If not found and tryGoPath=true, try lookInGoPath
       - If found, run version command and parse output
       - Return populated Dep struct

    Make all output-producing functions take io.Writer per ready.go pattern for testability.
    Tests use a test helper that creates temp directories with fake binaries (same pattern as fake_bd in graph package tests).
  </action>
