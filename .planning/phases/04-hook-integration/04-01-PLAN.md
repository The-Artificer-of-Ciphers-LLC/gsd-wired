    **Step 1: Create session_start.go (per D-03, D-04, D-05, D-06, research Pattern 4).**

    Create `internal/hook/session_start.go` with:

    ```go
    func handleSessionStart(ctx context.Context, raw json.RawMessage, hs *hookState, w io.Writer) error
    ```

    The function:
    1. Decodes raw into SessionStartInput.
    2. Fast path: checks if `.beads/` exists at `filepath.Join(input.CWD, ".beads")`. If not, emits hint: `"gsd-wired: No .beads/ directory found. Run /gsd-wired:init to initialize."` via writeOutput with AdditionalContext. Return immediately.
    3. Sets `hs.beadsDir = input.CWD` and calls `hs.init(ctx)`. On error, log to slog.Warn and return writeOutput with empty HookOutput (degraded mode per D-06).
    4. Creates a timeout context: 1500ms (leaving 500ms headroom within 2s budget).
    5. Calls `buildSessionContext(queryCtx, hs.client)` which returns a markdown string.
    6. Returns writeOutput with AdditionalContext set to the markdown.

    Create `buildSessionContext(ctx context.Context, c *graph.Client) string`:
    1. Query `c.QueryByLabel(ctx, "gsd:phase")` to find open phase beads. Find the one with highest `gsd_phase` metadata value and status "open" -- this is the current phase.
    2. Query `c.ListReady(ctx)` for ready tasks.
    3. Build markdown string (per D-03, D-04):
       ```
       ## GSD Project State

       **Current Phase:** {phase.Title} (Phase {N})
       **Status:** {phase.Status}

       ### Ready Tasks
       {for each ready task:}
       - [{task.ID}] {task.Title}
       {end}

       ### Phase Goal
       {phase.Description}
       ```
    4. On any query error, log to slog.Warn and return partial context (whatever succeeded). Never return error from buildSessionContext -- it always returns a string (possibly empty).
