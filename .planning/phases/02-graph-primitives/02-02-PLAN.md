         ```
           |-- Plan 02-01: bd CLI wrapper      [INFRA-03]
           +-- Plan 02-02: Domain mapping      [MAP-01, MAP-02]

         Total: N ready | M queued | P remaining
         ```
       - Use `|--` for non-last children, `+--` for last child (ASCII tree chars, not unicode — safer for all terminals)
       - Extract plan ID from `bead.Metadata["gsd_plan"]` (string value)
       - Extract requirement labels: filter `bead.Labels` for pattern matching `^[A-Z]+-[0-9]+$`
       - Counts: ready = len(ready tasks), queued = len(blocked tasks filtered for gsd:plan label client-side per Pitfall 6), remaining = ready + queued + in_progress
       - If no ready tasks: print "No ready work." and the queued/remaining counts
