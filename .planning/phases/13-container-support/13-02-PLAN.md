  <action>
    Create `internal/cli/container.go` following the Cobra subcommand pattern from setup.go (per D-01, D-03):

    1. **NewContainerCmd() *cobra.Command** — parent command "container" with Use "container", Short "Manage Dolt server container". Has two subcommands: start, stop.

    2. **NewContainerStartCmd() *cobra.Command**:
       - Flags: `--force` (bool, overwrite gsdw.compose.yaml), `--port` (string, default "3307"), `--beads-dir` (string, default ".beads/dolt")
       - RunE delegates to `runContainerStart(out io.Writer, opts startOpts)` for testability
       - `startOpts` struct: `force bool, port string, beadsDoltDir string, detectFn func(container.DetectOpts)(container.Runtime, error), composeFn func(string, container.ContainerConfig, bool)(string, error), execFn func(string, ...string) error, checkPort func(string) error, statFn func(string) error`

    3. **runContainerStart logic** (per D-03, D-04, D-05):
       a. Detect runtime via `detectFn` (default: `container.DetectRuntime` with default opts). On error, print guidance and return error.
       b. Print "Using runtime: {runtime.Name()}"
       c. Check `.beads/dolt/` exists via `statFn` (default: os.Stat). If missing: return error "`.beads/dolt/` not found — run `bd init --backend dolt` first" (per Pitfall 2)
       d. Check port availability via `checkPort` (default: `net.Listen("tcp", "127.0.0.1:{port}")` then close). If occupied: return error "Port {port} already in use — stop the existing Dolt server or use --port" (per Pitfall 3)
       e. Resolve absolute path for beadsDoltDir via `filepath.Abs`
       f. Build `ContainerConfig{BeadsDoltDir: absPath, HostPort: port}`
       g. For Docker/Podman runtimes: call `composeFn` to write gsdw.compose.yaml (per D-06). Print path to written fragment.
       h. Build start command from `runtime.StartArgs(cfg)`. Print the full command.
       i. Execute via `execFn` (default: exec.Command + Run). Print success or failure.
       j. Print: "Dolt container started on 127.0.0.1:{port}"
       k. Print: "Data persisted at {beadsDoltDir}" (per D-05, CNTR-07)

    4. **NewContainerStopCmd() *cobra.Command**:
       - RunE delegates to `runContainerStop(out io.Writer, opts stopOpts)`
       - `stopOpts` struct: `detectFn, execFn` (same injection pattern)

    5. **runContainerStop logic** (per D-03, CNTR-02):
       a. Detect runtime
       b. Build stop args from `runtime.StopArgs()`
       c. Execute
       d. Print "Dolt container stopped"

    6. **Port check helper**: `defaultCheckPort(port string) error` — attempts `net.Listen("tcp", "127.0.0.1:"+port)`, closes listener on success (port free), returns nil. Returns error if port occupied.

    Tests inject all dependencies (detectFn, composeFn, execFn, checkPort, statFn) for hermetic testing. No real containers, no real ports, no real filesystem checks in tests.
