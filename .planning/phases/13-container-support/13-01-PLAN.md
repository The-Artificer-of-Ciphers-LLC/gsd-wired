    Create `internal/container/runtime.go` with:

    1. **Runtime interface** (per D-12):
       ```go
       type Runtime interface {
           Name() string        // "docker", "podman", "apple-container"
           Binary() string      // resolved path to binary
           StartArgs(cfg ContainerConfig) []string  // args for exec.Command
           StopArgs() []string
           IsRunningArgs() []string
       }
       ```

    2. **ContainerConfig struct:**
       ```go
       type ContainerConfig struct {
           BeadsDoltDir string // absolute path to .beads/dolt/ on host
           HostPort     string // default "3307"
       }
       ```

    3. **DockerRuntime** (per D-04, CNTR-03):
       - StartArgs: `run -d --name gsdw-dolt -p 127.0.0.1:{hostPort}:3306 -e DOLT_ROOT_HOST=% -v {beadsDoltDir}:/var/lib/dolt dolthub/dolt-sql-server:latest`
       - StopArgs: `stop gsdw-dolt`
       - IsRunningArgs: `inspect --format {{.State.Running}} gsdw-dolt`

    4. **PodmanRuntime** — identical to Docker except:
       - Binary is `podman`
       - Port binding uses `0.0.0.0:{hostPort}:3306` (per Podman macOS Pitfall 6 from research)

    5. **AppleContainerRuntime** (per D-09, CNTR-05):
       - Binary is `container`
       - StartArgs: `run -d --name gsdw-dolt -p 127.0.0.1:{hostPort}:3306 -v {beadsDoltDir}:/var/lib/dolt dolthub/dolt-sql-server:latest`
       - No compose equivalent — Apple Container has no compose (per research)

    6. **DetectRuntime(opts DetectOpts) (Runtime, error)** (per D-02, D-13, CNTR-06):
       - `DetectOpts` has `LookPath func(string)(string,error)` and `MacOSVersion func()(int,int,error)` and `Arch func()string` for test injection
       - Priority: Apple Container (if macOS >= 26 AND arm64 AND `container` on PATH) -> Docker -> Podman
       - On macOS < 26: skip Apple Container silently (per D-10, only error when explicitly requested)
       - When none found: return `ErrNoRuntime` with install guidance

    7. **macOS version detection** (per D-09):
       - `defaultMacOSVersion()` runs `sw_vers -productVersion`, parses major.minor
       - Used by DetectRuntime default opts

    Tests use isolated PATH (same pattern as 12-01: `t.Setenv("PATH", dir)` with fake binaries). macOS version and arch injected via DetectOpts function fields for hermetic testing.
